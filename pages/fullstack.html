Fullstack Syllabus: Frontend + Backend

📖 6 modules + final capstone
1

Module 1: Frontend Foundations (HTML, CSS, JS)

📄 8 Lessons < > 5 Exercises ⬇️ 2 Challenges

Master modern HTML5, CSS3 (Flexbox, Grid, Animations), and JavaScript ES6+. Build responsive layouts, interactive components, and prepare for React.

🎨 Semantic HTML 💎 Tailwind CSS ⚡ JS DOM Manipulation
2

Module 2: React & Modern Frontend

📄 6 Lessons < > 4 Exercises ⬇️ 1 Challenge

Dive into React: components, props, state, hooks (useState, useEffect), context API, routing with React Router, and state management.

⚛️ React Components 🪝 Hooks & Context 🧭 React Router
3

Module 3: Backend with Node.js & Express

📄 7 Lessons < > 4 Exercises ⬇️ 2 Challenges

Learn server-side JavaScript with Node.js, build RESTful APIs using Express, handle HTTP requests, middleware, error handling, and environment variables.

🟢 Node.js Runtime 🚂 Express.js 🌐 REST API Design
4

Module 4: Databases (PostgreSQL & MongoDB)

📄 6 Lessons < > 3 Exercises ⬇️ 1 Challenge

Work with both SQL (PostgreSQL) and NoSQL (MongoDB) databases. Learn schema design, migrations, Mongoose ORM, and database integration with Node.js.

🐘 PostgreSQL 🍃 MongoDB 📊 Mongoose ODM
5

Module 5: Authentication & Security

📄 5 Lessons < > 3 Exercises ⬇️ 1 Challenge

Implement secure authentication using JWT, bcrypt password hashing, OAuth2, role-based access control, and protect routes against common vulnerabilities.

🔐 JWT Tokens 🛡️ Bcrypt 👥 RBAC
